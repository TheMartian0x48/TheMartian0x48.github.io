Trail Division

Introduction

Trail division is simplest algorithm to test whether a positive integer is prime or not. It is a brute force algorithm where we check whether the given number is divisible by and positive integer greater than 2 and less than itself.

Theorem 1

A positive integer $ n \gt 1 $ is a prime number iff for all positive integer $ d $, $ d \geq 2 $ and $ d \lt n $ , $ d \nmid n$.

Observation 1

In theorem 1, we are doing a lot of extra work. We can improve this vastly by using the fact that $d \geq \lceil \surd n \rceil $ make no useful contribution. Let's prove that.

Proof

Let k be positive integer such that $k \geq \lceil \surd n \rceil$ and $k \lt n$.

As $k * \lceil \surd n \rceil \gt n$ , $ k$ will divide $n$, that is $k * l = n$, if and only if $l \in [2, \lfloor \surd n \rfloor]$ . Since we already examining for all such possible $l$, it make no need to examine for such $k$.

End of proof

This observation leads to our second theorem.

Theorem 2

A positive integer $ n \gt 1 $ is a prime number iff for all positive integer $ d $, $ d \geq 2 $ and $ d \leq \lfloor \surd n \rfloor $, $ d \nmid n$.
