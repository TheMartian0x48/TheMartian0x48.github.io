Project Euler : Problem 28

Link

https://projecteuler.net/problem=28

Hints

Number of rows is increasing by 2 and difference between smallest diagonal entry in matrix with r rows is equal to largest diagonal entry in matrix with (r - 1) rows + (r - 1).

Source Code

[ C++ ]

#include <bits/stdc++.h>
using namespace std;

int main() {
  long long res = 1;
  int offset = 2, corner = 1;
  for (int i = 1; i < 1001; i += 2) {
    for (int j = 0; j < 4; j++) {
      corner += offset;
      res += corner;
    }
    offset += 2;
  }
  cout << res;
  return 0;
}
