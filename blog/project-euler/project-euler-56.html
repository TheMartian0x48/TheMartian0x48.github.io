Project Euler : Problem 56

Link

https://projecteuler.net/problem=56

Hints

Brute Force, Power

Source Code

[ Python 3 ]

res = 0
for i in range(1, 100):
  for j in range(1, 100):
    p = list(str(i ** j))
    res = max(sum([int(a) for a in p]), res)
print(res)
